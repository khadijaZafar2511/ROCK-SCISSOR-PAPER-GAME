# Timed Rock-Paper-Scissors Game

A classic Rock, Paper, Scissors game built using Vanilla JavaScript, HTML5, and CSS3, featuring a real-time countdown timer to test your speed against the computer.

## 🕹️ Gameplay & Features
*   **Dynamic 30s Timer**: Starts immediately when you click "Play", pushing you to score as much as possible before time runs out.
*   **Automated AI Opponent**: The computer makes completely randomized, fair choices simultaneously with your move.
*   **Real-time Scoreboard**: Keeps track of both User and Computer scores instantaneously during the match.
*   **Automatic Win Determination**: Evaluates the ultimate victor (User, Computer, or a Draw) precisely at the `0s` mark.
*   **Instant Reset**: The "Play Again" button clears scores, resets timers, and re-initializes the game state flawlessly.

## 🛠️ Tech Stack
*   HTML5 (Game Structure)
*   CSS3 (Layout & Visibility states)
*   JavaScript (ES6 Array Logic, Async `setTimeout` loops, DOM Event Handlers)
